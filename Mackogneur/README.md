## üìù Commands
### ‚ö† Note
> The default prefix is ':'  
> {arg} for the necessary arguments  
> (arg) for optional arguments

### All users

* Get information about commands.
     * `:help {@Pokemon}`
     * `:h {@Pokemon}`

* Get information about a user.
     * `:userinfo (ID or @)`
     * `:useri (ID or @)`
     * `:ui (ID or @)`

### Admins
* Get information about the bot.
     * `:ping {@Pokemon}`
     * `:p {@Pokemon}`
* Closes the server channels.
     * `:channelslock {on/off}`
     * `:chl {on/off}`

## üßæ Others features
